# 🍽️ Restaurant Food Ordering Backend API

A scalable RESTful backend for a Restaurant Food Ordering application built with **Node.js**, **Express.js**, and **MongoDB**.

## 🚀 Features

* User Authentication (JWT)
* Secure Password Hashing (bcrypt)
* Restaurant Management APIs
* Food & Category Management
* Order Management
* CRUD Operations
* Input Validation & Error Handling
* RESTful API Architecture
* API Testing with Postman

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt
* Postman

## 📌 API Base URL

```
http://localhost:8080/api/v1
```

## 📬 API Testing

All endpoints have been tested using **Postman**.
